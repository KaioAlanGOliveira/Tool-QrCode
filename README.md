#  Gerador de QR Code para WhatsApp

Aplicação desktop simples em Java que gera QR Codes para iniciar conversas no WhatsApp com mensagem pré-definida.

Basta informar o número de telefone, escrever uma mensagem (opcional) e gerar o QR Code. Ao escanear, a pessoa abre diretamente o chat no WhatsApp.

# Funcionalidades

- Seleção do país (DDI)
- Campo de telefone com máscara automática
- Mensagem personalizada (opcional)
- Geração automática do link `wa.me`
- Geração de QR Code em PNG
- Escolha da pasta e nome do arquivo
- Validações para evitar erros

# Como usar

1. Escolha o país (DDI)
2. Digite o número de telefone
3. Escreva a mensagem (se quiser)
4. Clique em **Montar link**
5. Defina o nome do arquivo, tamanho e pasta
6. Clique em **Gerar QRCode**

Pronto! O QR Code será salvo na pasta escolhida.
